HTML

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова
розмітки документів для перегляду вебсторінок у браузері.
Браузери отримують HTML документ від сервера за протоколами HTTP/HTTPS
або відкривають з локального диска, далі інтерпретують код в інтерфейс,
який відображатиметься на екрані монітора.

Парні теги

Парні теги складаються з відкриваючого та закриваючого тегів.
Вміст, що розміщується між цими тегами, визначає контент елемента.

Приклади: h1....h6, p, div, a

Непарні теги

Непарні теги (або самозакриваючі теги) складаються тільки з одного
тега і не потребують закриваючого тега. Вони зазвичай використовуються
для створення елементів, які не мають вмісту.

Приклади: br, hr, input...
